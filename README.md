# LobbyFacts DataThis is a repo for managing machine readable artifacts for the [LobbyFacts Data APIs](http://lobbyfacts.eu/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [LobbyFacts Data APIs](http://lobbyfacts.eu/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://lobbyfacts.eu/)- [Developer](http://lobbyfacts.eu/)The [APIs.json](https://github.com/api-evangelist/lobbyfacts-data/blob/master/apis.json) index provides links to all the relevant links for the LobbyFacts Data APIs, including the OpenAPI and Postman if they are present.
